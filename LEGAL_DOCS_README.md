# 📋 Rechtliche Dokumentation

Diese Datei enthält Informationen zu den rechtlichen Seiten, die für die öffentliche Nutzung der Website erstellt wurden.

## 🎯 Erstellte Seiten

1. **impressum.html** - Impressum & Kontaktinformationen
2. **datenschutz.html** - Datenschutzerklärung (DSGVO-konform)
3. **nutzungsbedingungen.html** - Nutzungsbedingungen
4. **cookies.html** - Cookie-Richtlinie

## ✅ Vor der Veröffentlichung anpassen

### 1. Impressum (impressum.html)
**WICHTIG:** Folgende Angaben müssen angepasst werden:
- `[Schulname]` - Name deiner Schule
- `[Schule]` - Schultyp (Gymnasium, Realschule, etc.)
- `[Adresse]` - Schuladresse
- `[Postleitzahl]` - Postleitzahl
- `[Stadt]` - Stadt
- `[Name]` - Name des Schulleiters/der Schulleiterin
- `[Email]` - E-Mail-Adresse der Schule
- `[Telefon]` - Telefonnummer der Schule

**Location:** Zeile 50-62

### 2. Datenschutzerklärung (datenschutz.html)
Diese Seite ist bereits DSGVO-konform konfiguriert.

Was wird gespeichert:
- ✅ Dark Mode Einstellung (localStorage)
- ✅ Cookie-Zustimmung (Cookies)
- ✅ Admin-Session (sessionStorage)
- ✅ Chatbot-Nachrichten (Backend)
- ✅ Besucher-Aktivitäten (Backend)

**Besonderheit:** Keine IP-Adressen oder persönliche Daten werden gespeichert.

### 3. Nutzungsbedingungen (nutzungsbedingungen.html)
Diese Seite kann so bleiben, ist aber an dein Projekt angepasst.

Wichtige Punkte:
- Keine kommerzielle Nutzung
- Nur für Bildungszwecke
- Chatbot-Warnung (KI-Antworten nicht immer korrekt)
- Haftungsbeschränkung

### 4. Cookie-Richtlinie (cookies.html)
Diese Seite ist bereits korrekt ausgefüllt.

Cookies auf dieser Website:

1. **Regelmäßige Updates:** Aktualisiere die Datenschutzerklärung, wenn sich Datenverarbeitung ändert

## 🚀 Deployment-Tipps

1. Überprüfe alle Schulkoordaten vor der Veröffentlichung
